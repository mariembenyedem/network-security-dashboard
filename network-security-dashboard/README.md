# Network Security Dashboard

This project is a practical cybersecurity and networking project that I built to better understand how network monitoring and intrusion detection systems work internally.

The main idea was to build a simplified system inspired by professional tools such as Wireshark and Snort, while learning how packet capture, traffic analysis, logging systems, APIs, and databases work together inside a real security monitoring architecture.

The project captures network packets in real time, analyzes traffic behavior, detects suspicious activity based on simple security rules, stores alerts, and exposes the results through a backend API and a monitoring dashboard.

---

## Why I Built This Project

As an IT student interested in networking, cybersecurity, and artificial intelligence, I wanted to build something more technical than a basic web application.

My goal was to understand how defensive security systems analyze network behavior and how security monitoring tools process traffic internally.

Instead of only learning theory, I decided to build a project that forces me to work with real networking concepts.

---

## Main Features

* Capture live network packets using Scapy
* Extract source IP, destination IP, and protocol information
* Count traffic generated by different IP addresses
* Detect suspicious traffic using threshold-based detection logic
* Store alerts in a local SQLite database
* Save packet logs in JSON format
* Expose monitoring data through a Flask API
* Display data through a simple dashboard interface
